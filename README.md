# Uber Utilities

iTerm 2 CLI for getting Uber estimates

```
$ brew install caskroom/cask/corelocationcli
$ CoreLocationCLI # confirm location access once
$ brew install yschimke/tap/uberalles

# Authorise with a Uber Oauth2 token
$ oksocial --authorize uber --token XXXXXXXXXXXXXXXXXXXXXX

$ ubereta destination
```
